# adfgvx
A simulation of the ADFGVX cipher used by Germany during WWI. This was pretty interesting to me because I was watching 1917 and thought about 
the ciphers germans maybe used before enigma/others. This is a js implementation of the ADFGVX cipher. 
If you're interested, you can learn more about it here https://en.wikipedia.org/wiki/ADFGVX_cipher.
